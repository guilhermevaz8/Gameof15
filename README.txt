JOGO DOS 15

COMO CORRER? 

1.Abrir o terminal e ir até ao diretório onde o jogo se encontra.

1.1 Se quiser jogar no teclado: 

            1.1.1 correr o ficheiro __init__.py

                1.1.1.1 Inserir configuração inicial seguida de enter e depois configuração final, pressionar j
                e começar a jogar.
            
1.2 Se quiser utilizar um método AI:

            1.2.1 correr o seguinte prompt "__init__.py 'método' <config.txt"

                1.2.1.1 Métodos disponiveis: BFS, DFS, IDFS, Greedy-misplaced, Greedy-Manhattan, A*-misplaced, A*-Manhattan.
                1.2.1.2 Qualquer alteração na configuração incial ou final deve ser efetuada no ficheiro config.txt

                
Testado em Python com a versão 3.8.10 Windows 11 Home 22H2.

Utilizamos ainda as bibliotecas: deque e heapq.
